Listas

Una lista puede contener varios elementos

frutas = ["manzana", "platano", "naranja"]

  • Están ordenadas y empiezan en el índice 0
  • Se pueden cambiar, añadir y borrar elementos de la lista
  • Permite elementos duplicados
  • Para saber el tamaño de la lista se usa la función len(nombreLista) len(frutas)
  • Se pueden mezclar tipos de datos en los elementos de la lista
  • La función type() devuelve: class 'list' type(frutas)
  • Para ordenar la lista se usa '.sort()' y ordena primero mayusculas y esta admite parametros como el 'reverse' frutas.sort(reverse = true)

    • def myfunc(n):
      return abs(n - 50)
      numeros = [100, 50, 65, 82, 23]
      numeros.sort(key = myfunc)


    • frutas.sort(key = str.lower)

  • Para copiar las listas se puede usar el '.copy()' y asi consigues dos listas diferentes, si no creas una copia y le das el valor de una a otra varible se cambiaran ambas.
    frutasCopia = frutas.copy()
  • Se pueden concatenar listas con un '+' y la guardamos en una nueva lista.
    listaConcat = frutas + numeros

nuevaLista = [ 'x' for 'x' in frutas if 'x' != "manzana"

Tuplas

Una tuple puede contener varios elementos

frutas = ("manzana", "platano", "naranja")

  • Están ordenadas y empiezan en el índice 0
  • No se pueden modificar, añadir y borrar elementos
  • Permite elementos duplicados
  • Para saber el tamaño de la tupla se usa la función len(nombreTupla) len(frutas)
  • Se pueden mezclar tipos de datos en los elementos de la lista
  • La función type() devuelve: class 'tuple' type(frutas)
  • Se puede desempaquetar tuplas, tal que asignas a varias variables los objetos de la tupla de manera ordenada
    (verde,amarillo,naranja) = frutas
    verde = manzana
    amarillo = platano
    naranja = naranja

Sets

Un set puede contener varios elementos

frutas = {"manzana", "platano", "naranja"}

  • No estan ordenados
  • No se pueden modificar, pero si se pude añadir y borrar elementos
  • No se permiten elementos duplicados
  • 'True' y '1' es lo mismo
  • Se pueden mezclar tipos de datos en los elementos del set
  • La función type() devuelve: class 'set' type(frutas)

Diccionarios

Un diccionario es un almacen de informacion en formato clave valor

frutas = {"clave1":"manzana", "clave2":"platano","calve3":"naranja"}

  • Son ordenados
  • Se pueden cambiar, añadir y borrar valores del diccionario
  • No se permiten claves duplicadas
  • Se pueden mezclar tipos de datos en los valores de un diccionario y las claves son 'str'
  • La función type() devuelve: class 'dict' type(frutas)
